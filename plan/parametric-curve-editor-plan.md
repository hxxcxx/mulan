# 参数曲线编辑器完善计划

## 当前判断

Bezier / B 样条 / NURBS 已经能画、能选、能显示控制点、能拖控制点，但现在仍然只是“基础可编辑”。下一步最值得推进的是让它们进入真正的 CAD 编辑对象层级。

## 控制点编辑完整化

- 选中控制点后支持删除控制点。
- 在曲线上插入控制点。
- 多选控制点后整体移动。
- 控制点拖动时，snap / 轴约束继续生效。

## B 样条 / NURBS 专属编辑

- B 样条支持节点插入，而不是简单加控制点。
- NURBS 支持权重编辑，否则权重全为 1 时视觉上基本等同于 B 样条。
- 后续补 degree / knot mode 属性。

## 捕捉系统补齐曲线语义

- 捕捉控制点。
- 捕捉曲线最近点。
- 捕捉曲线端点、中点、等分点。
- 为未来面、放样、扫掠、体生成提供可靠输入。

## 下一步实施目标

下一步做：参数曲线控制点编辑完整化 + NURBS 权重编辑的基础框架。

完成这一步后，参数曲线就不只是“能画”，而是开始具备可持续扩展的 CAD 编辑语义。
